## POTENTIAL BREAKING CHANGES

### 1. XMLUnit 1.3 → 2.10.0 (MAJOR API CHANGE)
XMLUnit 2.x completely rewrote the API. Tests using XMLUnit will need updates.

Old API (XMLUnit 1.x):
  XMLAssert.assertXMLEqual(expected, actual);

New API (XMLUnit 2.x):
  import static org.xmlunit.matchers.CompareMatcher.isSimilarTo;
  import static org.hamcrest.MatcherAssert.assertThat;
  assertThat(actual, isSimilarTo(expected));

**Action Required**: Update all test code using XMLUnit to new API.

### 2. JAXB 3.0 → 4.0 (Jakarta EE 11)
- Package namespace remains `jakarta.xml.bind`
- Minor API enhancements, mostly backward compatible
- Schema validation improvements

**Action Required**: Test XML marshalling/unmarshalling, especially custom adapters.

- Performance improvements

- Enhanced criteria API
- Minor deprecations

**Action Required**: Test ORM queries and entity mappings.
