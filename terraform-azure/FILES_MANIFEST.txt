QUICK START
===========

1. Copy example variables:
   cp terraform.tfvars.example terraform.tfvars

2. Edit with your values:
   nano terraform.tfvars

3. Initialize:
   terraform init

4. Deploy:
   terraform apply

Or use the automated script:
   ./scripts/deploy.sh dev
