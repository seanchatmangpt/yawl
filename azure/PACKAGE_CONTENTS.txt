QUICK START (5 MINUTES)
================================================================================

1. PREREQUISITES
   ✓ Azure CLI 2.30+ installed
   ✓ Azure subscription with appropriate permissions
   ✓ Local machine with bash or PowerShell

2. AUTHENTICATE
   $ az login
   $ az account show

3. DEPLOY
   $ cd /home/user/yawl/azure
   $ chmod +x deploy.sh
   $ ./deploy.sh production eastus
   (Wait 45-60 minutes for deployment to complete)

4. VERIFY
   $ az deployment group show --resource-group yawl-production \
     --query properties.outputs

5. ACCESS
   Open browser: https://<app-service-name>.azurewebsites.net/resourceService/
