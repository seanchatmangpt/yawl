## What is YAWL DSPy?

YAWL DSPy integrates [DSPy](https://dspy.ai/) (Declarative Self-improving Language Programs in Python) with the YAWL workflow engine via GraalPy. This enables:

- **ML-powered workflow decisions** - Use optimized language models for worklet selection
- **Predictive resource routing** - AI-driven task allocation
- **Anomaly forensics** - Intelligent root cause analysis
- **Runtime adaptation** - Autonomous workflow modification

## Key Features

- **GEPA Optimization** - Gradient Estimation for Prompt Architecture
- **MCP Tools** - Model Context Protocol for LLM integration
- **A2A Skills** - Agent-to-Agent protocol for autonomous systems
- **Behavioral Footprints** - Perfect workflow generation validation
- **Hot Reload** - Update DSPy programs without restart

## Requirements

- Java 25+
- Python 3.11+ (via GraalPy)
- YAWL Engine 6.0.0+
- DSPy 2.5+
